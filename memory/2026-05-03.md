### 关键教训总结（2026-05-03）

1. **PowerShell + 中文HTML = 损坏**：永远不用 `Get-Content | Set-Content -NoNewline` 处理含中文的HTML/Python，改用 Python `Path.write_text()` 直接写入

4. **DBeaver 驱动属性**：`allowPublicKeyRetrieval=true` 和 `useSSL=false` 是国内 MySQL 连接标配
5. **文本写入**：Windows 环境写含中文的文本文件（CSV/SQL/HTML）优先用 Python 或 qclaw-text-file skill，避免 PowerShell 编码问题
